2. Verify exact TMDB page manually.
3. Add `tmdbId`.
4. If the fix changes poster/cast/synopsis for users, bump `CACHE_VERSION` in
   `docs/assets/src/tmdb-client.js`.
5. Run:

```sh
ruby tools/validate-catalog.rb
```

6. Review the remaining missing list before shipping.
